TitledSupplementaryViewConfig

public struct TitledSupplementaryViewConfig<Item> : ReusableViewConfigProtocol

A TitledSupplementaryViewConfig is a specialized supplementary view config that conforms to ReusableViewConfigProtocol. This config is responsible for producing and configuring TitledSupplementaryView instances.
