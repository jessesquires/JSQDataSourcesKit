ViewFactory

public struct ViewFactory<Item, Cell: ReusableViewProtocol>: ReusableViewFactoryProtocol

A ViewFactory is a concrete ReusableViewFactoryProtocol type. This factory is responsible for producing and configuring reusable views for a specific item. Cells can be for either collection views or table views.

    A unique identifier that describes the purpose of the cells that the factory produces. The factory dequeues cells from the collection view or table view with this reuse identifier.
